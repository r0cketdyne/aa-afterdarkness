jQuery: the 'J' is for 'Victory'

Selector

  • tag: $('tag')
  • id: $('#idName')
  • class: $('.className')
  • combo: $('.className:lastChild')

Manipulator

  • css class: $(selector).addClass('class')
  • removal: $(selector).detach()
  • add after: $(selector).insertAfter(htmlElement)
  • add after parent: $(selector).parent().append(htmlElement)

Eventor

  • initial script: $(document).ready(func)
  • click: $(selector).on("click", fun)
  • submit: $(selector).on("submit", fun)
  • off: $(selector).off("eventType", fun)
