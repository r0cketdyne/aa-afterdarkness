To set up the demo database, in terminal run:

* `psql`
* `CREATE DATABASE lecture;`
* `\q`
* `cat demo/setup_lecture_demo.sql | psql lecture`

Now you can open up the lecture database with:
* `psql lecture`
* `\d`
